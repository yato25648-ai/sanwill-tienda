# 🥤 SANWILL - Tienda de Bebidas Tradicionales

## Descripción del Proyecto

**SANWILL** es una tienda en línea moderna para la venta de bebidas tradicionales y refrescantes. Nuestro catálogo incluye:

- **Limonada** - Refrescante bebida de limón natural
- **Lulada** - Bebida tropical hecha de lulo
- **Mango Biche** - Bebida refrescante de mango verde
- **Maracuyada** - Deliciosa bebida de maracuyá
- **Salpicón** - Bebida tradicional con frutas

## Características

✅ **Diseño Responsivo** - Se adapta a dispositivos móviles, tablets y computadoras
✅ **Catálogo Interactivo** - Visualización clara de productos con imágenes
✅ **Interfaz Moderna** - Navegación intuitiva y fácil de usar
✅ **Información Completa** - Descripción detallada de cada bebida
✅ **Optimizado para SEO** - Mejor visibilidad en motores de búsqueda

## Estructura del Proyecto

```
sanwill_tienda/
├── index.html          # Página principal de la tienda
├── images/             # Carpeta con imágenes de bebidas
```

## Cómo Funciona

1. **Visualización de Productos**: La página principal muestra todos los productos en cards interactivas
2. **Información Detallada**: Cada producto incluye imagen, nombre, descripción y precio
3. **Contacto**: Información de contacto para realizar pedidos
4. **Diseño Responsivo**: Se ajusta automáticamente a cualquier tamaño de pantalla

## Desarrollo Local

Para ejecutar el proyecto localmente:

```bash
# Opción 1: Usar Python
python3 -m http.server 8000

# Opción 2: Usar npm
npm start
```

Luego abre tu navegador en `http://localhost:8000`
