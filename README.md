# 🎬 tuplataforma - Tienda de Streaming

## Descripción del Proyecto

**tuplataforma** es una tienda en línea para venta de cuentas y combos de plataformas de streaming en Colombia.

Catálogo actual:

- Netflix Premium
- Disney+
- HBO Max
- Spotify Premium
- Crunchyroll
- YouTube Premium
- Universal+
- Combo 2 plataformas
- Combo 3 plataformas
- Combo 4+ plataformas

## Características

✅ Diseño responsivo
✅ Catálogo visual con tarjetas de productos
✅ Selector de plan (Básico/Premium)
✅ Cantidades por producto
✅ Carrito flotante con total en COP
✅ Envío de pedido por WhatsApp

## Estructura del Proyecto

```
tuplataforma_tienda/
├── index.html          # Página principal
├── images/             # Imágenes y referencias visuales
├── package.json        # Configuración del proyecto
├── vercel.json         # Configuración para Vercel
├── .gitignore
└── README.md
```

## Desarrollo Local

```bash
python3 -m http.server 8000
# o
npm start
```

Abrir en `http://localhost:8000`.
